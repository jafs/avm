Requisitos
==========
Pentium III (o compatible) 500 MHz o superior.
Windows XP o superior.
.NET Framework 3.5 o posterior.


Descripción
===========
AVM (Android Virtual Management) te permite controlar máquinas virtuales de
Android de forma gráfica. Permitiendo cambiar el estado de la batería, sensores,
GPS, etc. de una forma sencilla.

Instalación
============
Si has descargado un fichero .zip, descomprímelo en cualquier directorio de tu
disco. Verás cuatro ficheros:

 - El fichero ejecutable de la aplicación llamado 'avm.exe'.
 - Un fichero con el contenido de la licencia: license.txt.
 - Este fichero LÉAME en inglés y español.


Cómo iniciarlo
==============
Haz doble clic sobre el icono del ejecutable en el explorador de ficheros, o
escribe "avm" bajo el prompt.
